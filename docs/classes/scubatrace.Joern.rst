Joern
=====

Joern is a code analysis platform. ScubaTrace's Joern module provides a Python interface to interact with the Joern code property graph (CPG) database. It allows users to query and analyze code properties using a high-level API, making it easier to perform complex code analysis tasks. Meanwhile, ScubaTrace utilizes Joern's capabilities to enhance call graph generation and code property analysis, enabling more accurate and comprehensive insights into codebases.
